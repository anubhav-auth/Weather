## Overview

**WeatherApp** is a sleek and modern Android application that provides accurate and up-to-date weather information for any location. Developed using Kotlin and Jetpack Compose, this app offers a seamless and intuitive user experience, allowing you to stay informed about the weather wherever you are.

### 1. Install the App
- [Download the latest version](https://github.com/anubhav-auth/Weather/releases) from the releases section.

### 2. Search Your Location
- Open the app and search for your desired location(s).

### 3. Monitor Weather
- Get real-time weather updates and view the 7-day forecast.
